# WOW Mr.Beast Audio Player

点击系统托盘图标即可播放 WOW Mr.Beast 音频。

## 功能特点

- 🎵 **一键播放**：点击托盘图标立即播放音频
- 📊 **无 UI 界面**：后台运行，不占用桌面空间
- 🔄 **多实例支持**：可同时运行多个实例，独立播放
- 🎮 **系统托盘**：右下角任务栏图标，随时控制

## 系统要求

- Windows 7 / 8 / 10 / 11
- .NET Framework 3.5 SP1 或更高版本
- 支持 WAV 格式的音频文件

## 使用方法

1. 确保已安装 [.NET Framework 3.5 SP1](https://dotnet.microsoft.com/download/dotnet-framework/net35-sp1)
2. 将 `WOW-MrBeast.exe`、`Main.wav`和`Main.ico` 放在同一目录
3. 双击运行 `WOW-MrBeast.exe`
4. 系统托盘会出现图标，点击左键即可播放音频


## 编译方法

直接运行build.bat


## 多开说明

支持同时运行多个实例，每个实例独立控制播放，互不影响。
